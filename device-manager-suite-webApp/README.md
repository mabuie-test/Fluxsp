# device-manager-suite

Este repositório inclui:

- `web-frontend/`: interface web (mantida sem alteração de UI/fluxos).
- `backend/`: backend legado em Node.js + MongoDB (referência).
- `backend-php/`: backend convertido para PHP + MySQL, compatível com os mesmos endpoints `/api`.

Para executar a versão convertida, ver `backend-php/README.md`.
